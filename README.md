# claude-code-plugins

My personal Claude Code plugin marketplace.

## Install

```
/plugin marketplace add brayanobisto/claude-code-plugins
/plugin install flow-verify@claude-code-plugins
```

Update later with `/plugin marketplace update claude-code-plugins`.

## Plugins

- **[flow-verify](plugins/flow-verify/)** — author and run natural-language
  browser-verification flows. Requires the `agent-browser` skill.
